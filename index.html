Qu'est-ce qu'un Terminal Unix ?


  • Terminal Unix
    • 2.Quelques commandes de bases.

  • Le terminal unix est un programme qui ouvre une console dans une interface graphique, il permet de lancer des commandes.

  • Il est parfois plus simple de taper une commande que d'effectuer des manipulations demandant beaucoup de clics de souris dans une interface graphique. C'est aussi un moyen plus simple pour expliquer comment faire quelque chose à quelqu'un, puisque il suffit d'indiquer la commande et non la suite de clics à effectuer sur l'interface graphique.
